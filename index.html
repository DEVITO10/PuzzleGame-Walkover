Help

Are you stuck ? Maybe this can Help
You can access this Help section at any moment. Have Fun Playing.

  • Choosing a game:
    You can only choose one game at a time. Click/Press on the button and the game will start-up.

  • Exiting from a game:
    If to change the game or just want to go to the home screen, click/press on the home button given at the top right corner.
    You can locate it just beside the help button.

  • Rules of the Game:
    For every game the playing rules are given on the right side of the main playing board.
    Read them carefully before starting the game.

  • Tic-Tac-Toe
    This is a two player game.
    • The Player 1 starts first as 'X' followed by Player 2 as 'O'.
      If confused about the turn, the 'Turn' counter will display which player's turn it is.
    • When any of the player wins the game will stop and the winner's name will pop-up on the right side of the board.
    • As soon as any player wins the game will stop and you will have to start another game using the 'Replay' button.
      'Replay' button is present at the bottom of the right side of the board.

  • Sudoku
    This is a one player game.
    There can be many ways to solve this Sudoku, but there is only one correct way to solve this Sudoku.
    • As soon as you correctly solve the Sudoku the game will stop and you will be declared as winner.
    • You can reset or create a new board using the 'Replay' button given on the right bottom side the board.
